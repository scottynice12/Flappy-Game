How to play:
Click space to start flying
Click the restart button at the bottom

Save the file – Copy the entire code above and save it as index.html (exact lowercase name).
Repository setup – Make sure your GitHub repository has this file in the root folder (not inside a subfolder).
Enable Pages – Go to your repo → Settings → Pages → Set source to main branch (or master) and /root folder.
Wait 1-2 minutes – Your game will be live at:
https://your-username.github.io/your-repo-name/
🎮 Game Features
Tap, click, or press Space to make the bird fly
Avoid the green pipes
Score increases when you pass each pipe pair
High score is saved in your browser (localStorage)
Works on mobile and desktop
Once you add this file as index.html in the root of your repository, your game will load correctly.
